Display and Positioning

Concept Description
Position Specifies the position of an element on the page.
Relative An element’s position is relative to its default position on the page.
Absolute An element’s position is relative to its closest positioned parent element. It can be pinned to any part of the web page but will move with the document during scrolling.
Fixed An element’s position can be pinned to any part of the web page and will stay in view when the page is scrolled.
Sticky An element sticks to a defined offset when the user scrolls its parent container.
Z-Index Specifies how far back or forward an element appears when overlapping other elements.
Display Controls how an element flows vertically and horizontally within the document.
Inline Inline elements take up minimal space and cannot have manually adjusted width or height.
Block Block elements take up the full width of their container and can have manually adjusted heights.
Inline-block Inline-block elements can have set width and height, and can appear next to each other without occupying the full container width.
Float Moves elements as far left or as far right as possible on the page.
Clear Clears an element’s left or right side (or both) of floating elements.
